Scholarly is a student tracking and management system designed to help students and administrators manage academic records efficiently. It allows students to view and print their academic history while providing admins with tools for student record management.

Bloppy Blord is a Flappy Bird-inspired game where players tap to keep their character airborne while avoiding obstacles. Unlike traditional endless runners, the game ends when you reach a set score, making each run a race to the finish!
